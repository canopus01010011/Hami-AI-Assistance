# Hami-AI-Assistance
Hami AI - An intelligent hamster-themed AI assistant built with React, TypeScript, FastAPI, Gemini API, and SQLite. Hami can chat, understand user intentions, manage tasks, deadlines, and display animated moods.


# 🐹 Hami AI

Hami AI is a hamster-themed personal AI assistant built with:

- React + TypeScript
- FastAPI
- Gemini API
- SQLite

Hami can:

✅ Chat naturally with users

✅ Detect user intentions

✅ Save tasks and deadlines

✅ Retrieve saved deadlines

✅ Display animated hamster moods

✅ Act as a personal productivity assistant
